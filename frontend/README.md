# ShopSmart Frontend

The frontend is a responsive React socks ecommerce UI built with Vite. It includes:

- Shopper storefront (catalog, cart, checkout)
- Admin panel (product CRUD, order status management)

## Commands

```bash
npm install
npm run dev
npm run lint
npm test
npm run build
```

Set `VITE_API_URL` in `frontend/.env` if your backend is not running on `http://localhost:5001`.
